🎨 概要

Ink Labyrinth は、
「黒い迷路の中をインクトレイルを描きながら進む」
シンプルで綺麗なタッチ操作ゲームです。

迷路は毎回ランダム生成

白い通路の上だけ描画できる

インク量は移動距離で減少

ゴール（出口）に到達すればクリア

壁に触れるかインク切れでゲームオーバー

キャンバス 2 枚を重ね、
1枚目に迷路、2枚目にインクトレイルを描く構造になっています。

🕹️ 遊び方

マウスまたは指で画面をなぞるとインクが描かれる

白い通路の上だけ描ける

ゴール（右下の黄色い小さなマーカー）へ到達するとクリア

インクが 0% になるとゲームオーバー

壁（黒）に触れると即ゲームオーバー

「再生成」ボタンで迷路を作り直せます

🗺️ 仕組みの説明（簡潔）
1. キャンバス構造

mazeCanvas：迷路の描画（白が通路、黒が壁）

drawCanvas：インクトレイルの描画

迷路の画素を getImageData() で読み取り、
通路（白）か壁（黒）かを判定しています。

2. 迷路生成

再帰的バックトラッカー（depth-first search）による
ランダム迷路生成アルゴリズムを使用。

迷路のサイズは難易度により変化（Easy / Normal / Hard）

開いた壁を四方向で管理（north / east / south / west）

3. インクの仕組み

ポインタの移動距離に応じてインクが減る

INK_DRAIN_PER_PX を小さくして描きやすく調整済み

インクトレイルの太さは残りインク量で少し変化

4. 当たり判定

迷路キャンバスのピクセルを読み取り

白なら進める / 黒なら壁判定

1px 単位で正確に壁判定を行っています
